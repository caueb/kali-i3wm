# kali-i3wm
Kali with i3wm-gaps

# Dependencies
## Apps
- polybar (Top bar)
- rofi (Run apps)
- kitty (Terminal)
- Atom (Text editor)
- Thunar (File manager)
- Flameshot (Screenshot)
